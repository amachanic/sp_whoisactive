# sp_WhoIsActive (2012 - 2019)

SQL Server 2012 - 2019 compatible version.

This version includes fixes and features that require SQL Server 2012+ (e.g. `TRY_CAST`), but avoids features only available in SQL Server 2022 or later.

For SQL Server 2005 - 2008, use the [`/2008`](../2008/) folder. For the latest version, use the [current version](../) in the root folder.
