# sp_WhoIsActive (2005 - 2008)

SQL Server 2005 - 2008 compatible version.

This version avoids features unavailable before SQL Server 2012 (e.g. `TRY_CAST`, newer DMV columns). Use this if you are running SQL Server 2005, 2008, or 2008 R2.

For SQL Server 2012+, use the [`/2019`](../2019/) folder or the [current version](../) in the root folder.
